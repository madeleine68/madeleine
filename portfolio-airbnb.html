Airbnb Data Sets Integrated into Tableau Dashboard

Project information

  • Category: Tableau Dashboard

The Dashboard in Tableau enables data to be presented in a dynamic manner. Dashboards are collections of many views. Through them, you are able to compare various data types concurrently. Rather than having to navigate to separate worksheets, you can create a dashboard that displays all the views from various perspectives at once. This essay will walk you through an example demonstrating how to create an interactive Data Dashboard with Airbnb datasets using Tableau.
